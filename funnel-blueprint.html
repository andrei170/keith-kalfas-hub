Tech Stack (recommended, swappable - keeps Keith's existing stack where possible)

  • CMS / Cohort host: Kajabi (Keith already uses Kajabi for the Academy, the Alliance, and existing courses - the cohort plugs in as a new Kajabi cohort product). Skool or Circle as the community channel if the Alliance currently uses one.
  • Landing pages: Kajabi-native (fast, integrates with Keith's existing email + course stack) or custom HTML on the AG1 side for the cohort landing if Kajabi templating is too constraining
  • Email: Kajabi-native (keeps the email stack consolidated with the Alliance + courses) or Kit / ConvertKit if a deeper segmentation layer is wanted
  • Quiz / Scorecard: ScoreApp (preferred for 0-100 scoring + trap-type segmentation logic) or Typeform with Logic Jumps
  • Payment: Stripe (cohort) + Kajabi-native for the existing Alliance and courses
  • Video host: Wistia (cohort landing) + YouTube (long-form continues to live there) + Loom (1:1 audit calls in cohort Premium tier)
  • Analytics: GA4 + Meta Pixel + Microsoft Clarity (heatmaps on landing)
  • CRM: Kajabi-native CRM (keeps the cohort + Alliance + bundle data in one place) or HubSpot Starter if a real CRM is wanted
  • Community / Cohort Channel: Skool (low overhead, mobile-friendly, what most trades cohorts use) or Circle. Keith's existing Alliance community gets a separate Skool space for cohort actives.
  • Scheduling: Calendly (default for office hours) or Cal.com
  • Project / build: Notion (handover-friendly) + Linear (engineering)

Integration Points

  1. Scorecard → Email · ScoreApp triggers Kit / Kajabi-native on quiz completion with score tag + trap-type tag + stage tag
  2. Email → Cohort · Kit / Kajabi triggers cohort enrolment on Stripe purchase webhook
  3. Email → Alliance · same trigger, different product webhook
  4. Cohort → 1:1 Coaching Call · cohort grads on Premium tier get a Calendly link routed to Keith's existing coaching call calendar with cohort-grad tag attached
  5. Cohort → Alliance renewal · cohort grads get an Alliance discount code on cohort completion (Alliance retention lift)
  6. Stan-style / coaching call confirmation page → Cohort waitlist · 1:1 call bookers who are cohort-fit get added to the cohort waitlist
  7. All events → GA4 + Meta Pixel via GTM · captures full funnel attribution
  8. Podcast end-card + show-notes link → scorecard with UTM tag ?utm_source=podcast&utm_medium=show-notes&utm_campaign=trap-score
  9. YouTube end-card + description link → scorecard with UTM tag ?utm_source=youtube&utm_medium=end-card&utm_campaign=trap-score

Owner Map

  • Keith: filming (VSL + photoshoot day + 6-episode cohort mini-series), weekly group call on the live cohort cycle, scheduled office-hours blocks (2x monthly), top-line decisions on copy + brand, mid-roll podcast reads for next 10 episodes
  • AG1 (Andrei): everything else - landing page build, scorecard setup, email writing, automation, cohort onboarding flow, weekly reporting, ads management, YouTube end-card refresh on top 20 evergreens, show-notes template refresh

Failure Modes + Mitigations

  • VSL filming delays → use existing YouTube long-form (zpnOhq67U20 "How to Start a Landscaping Business with NO Money | 50% Profit Margins" or tOUv60GebJQ "10,000 Landscaping Business Startups") as Phase 1 placeholder; replace with custom cohort-specific VSL by end of Phase 1
  • Kajabi templating constraint on the cohort landing → AG1 builds a custom HTML landing on the AG1 side with the Kajabi cohort enrolment link embedded, redirects from Kajabi when bandwidth opens
  • Cohort cap of 20-25 filling slowly on first cycle → run a smaller pilot cohort (12 seats) at the standard tier price to lock the format before opening cap to 25
  • Alliance cannibalisation concern → frame the cohort explicitly as "the implementation upgrade for Alliance members" and route cohort grads back to the Alliance at a renewal discount - the two are complementary, not competing
  • 1:1 coaching call concern → frame the 1:1 as the cohort-grad premium continuation - cohort grads on Premium tier get priority 1:1 bookings, which protects the high-touch lane while feeding it
  • Podcast end-card reads cross legacy episodes → only update next 10 forward episodes; do not back-edit hundreds of old episodes - the show-notes template refresh handles the back-catalogue passively
  • Scorecard answer fatigue → 12 questions is the cap. Test 8-question version against 12-question version after 30 days; pick whichever converts higher email opt-in

KPIs (90-day)

  • Phase 1 · Landscaping Operator Cohort landing live · 12-25 cohort applications in first 30 days from existing audience
  • Phase 2 · 1,500+ scorecard completions in first 90 days · 5-8% IG-to-email conversion from the existing 36K IG base
  • Phase 2 · Podcast end-card refresh on next 10 episodes · 800+ scorecard completions attributed to podcast in first 90 days
  • Phase 3 · 6-episode Cohort Module Walkthrough mini-series shipped · 40+ short-form posts published from the hook bank
  • Phase 4 · Meta CPL under $15 for scorecard opt-in · cohort CAC under $400 (mid-tier price band absorbs $200-500 CAC comfortably)
  • Year-1 floor target · $300K-$960K incremental on top of existing Alliance + courses + 1:1 coaching call revenue
